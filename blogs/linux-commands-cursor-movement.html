# 终端光标移动

# 前移一个词

alt + f （forward），mac 中 option（alt）+ 方向右键

# 后移一个词

alt + b （back），mac 中 option（alt）+ 方向左键

# 移动至命令行首

ctrl + a

# 移动至命令行尾

ctrl + e （end）

# 移动到当前词的结尾

esc + f

# 移动到当前词的开头

esc + b
