# 终端控制

# 打开终端

ctrl + alt + t

# 关闭终端

ctrl + shift + w

# 新建窗口

ctrl + shift + n（new）

# 新建 tab

ctrl + shift + t（tab）

# 在 tab 之间切换

  • alt + tab 的编号（如 alt + 2， 切换到第二个 tab）
  • ctrl + PgUp 切换到上一个 tab
  • ctrl + PgDn 切换到下一个 tab

# 关闭 tab

ctrl + d
